How to Use This Website

  • Home Page: Welcome to the fashion hub with the latest clothing trends – click "Shop Now" to explore the collections.
  • About Page: Learn about the creator of the site and the purpose of this fashion project.
  • Gallery Page: Enjoy stylish clothing photos along with a table showing prices and categories.
  • Booking Page: Fill out the form to book an appointment – include your name, gender, fashion interest, and comments.

Tips

  • All pages are connected using the navigation bar at the top.
  • All forms have required fields – make sure you fill them before submitting.
  • Images have alternative text for accessibility.
  • The website is built using HTML5 & CSS3 with Flexbox layout.
